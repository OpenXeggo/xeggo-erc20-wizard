# xeggo-erc20-wizard
Create ERC20 tokens without code!


SDK link: https://github.com/OpenXeggo/Xeggo-drop-server

# How to run:
This porject can be run by hostign the fiels inside the repo via any static server. 
We at Xeggo use [Serve](https://www.npmjs.com/package/serve "Serve") for local deployment.

### Installation
```
npm i -g serve
OR
yarn global add serve
```

### To Run
```
serve
```
Navigate to [http://localhost:3000](http://localhost:3000 "http://localhost:3000")
